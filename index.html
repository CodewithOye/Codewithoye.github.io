What I do

UX/UI Design

I create visually appealing and user-friendly designs for websites, considering user experience (UX) and user interface (UI) principles. This involves creating wireframes, mockups, and prototypes using tools such as Sketch, Figma,canvas or Adobe XD.

Front-end Development

I code responsive and interactive web pages using HTML5, CSS3, and JavaScript, ensuring cross-browser compatibility and accessibility. I also have experience with front-end frameworks such as Bootstrap, Foundation, and Bulma.

I ensure that websites are optimized for different devices and screen sizes, using responsive design techniques and media queries, so that the website looks great on desktops, tablets, and mobile devices.

I work closely with clients, designers, and other stakeholders to understand project requirements, provide regular updates, and ensure that the project is completed successfully and on time.

Continuous Learning

I stay updated with the latest web design and development trends, tools, and technologies to enhance my skills and provide cutting-edge solutions to clients.
